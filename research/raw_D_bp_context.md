### 目标用户定义

**核心用户群：一人公司创业者（Solo Entrepreneur / OPC）**

| 属性 | 描述 |
|------|------|
| 规模 | 个人创业者，无全职员工或极少 |
| 技术背景 | 非技术出身（内容创作者、咨询顾问、设计师、自媒体等） |
| 使用场景 | 内容创作、数据分析、项目管理、市场调研 |
| 核心痛点 | 独自承担所有工作职能，AI 工具分散、协作低效 |
| 期望感受 | 掌控感、专业感，不被技术术语吓到 |

**市场规模数据**
- 全球自由职业者和独立创业者：**15.7 亿**人（DemandSage, 2025）
- 美国 Solopreneur：**4,180 万**人（2026）
- 新创业公司中单人创始占比：**36.3%**（2025 H1，vs 2019 年 23.7%）
- Solopreneur 年工具支出（全栈）：**$3,000–$12,000**
- 中国个体工商户：**1.27 亿**（国家市场监管总局，2025）
- 1-9 人公司的 AI 投资率仅 **24%**，而 50-249 人公司为 **75%**（不是不想用，是用不了）

**核心痛点排序**
1. 多 AI 工具之间无法协作，结果需手动传递（最痛）
2. 复杂任务无法分解给不同 AI，只能单线程（明显）
3. 工作流无法保存和复用，每次重头配置（明显）
4. AI 没有"持续记忆"，每次对话断层（中等）
5. 不懂技术，无法使用 n8n / Dify 等专业工具（巨大）

**用户心声引用**
- 87% 的用户将"AI 记住我的偏好"列为最重要的功能需求（Index.dev, 2025）
- 每周浪费 5-8 小时在工具上下文切换上（Business.com 报告）

### 核心价值主张

**品牌名**：Solarian（原 AgentFlow，2026-04-14 正式定名）
**域名**：solarian.app

**一句话定位（两个版本）**
- 英文：**"Solarian — the agent civilization"**
- 中文：**"索拉利安——硅基员工文明"**

**产品定义**
- **"一人公司的 AI 团队操作系统"**
- **"你的硅基团队工厂"**
- **"一个人 + 一支 AI 团队"**

**三层产品逻辑**
1. **持久化的硅基员工**：每个 Agent 有独立身份、记忆与技能库，长期存在，越用越专精
2. **可复用的工作流**：打磨一次，永久复用，多 Agent 协作端到端交付结果
3. **自进化的系统**：Agent 自己写 Skill、提炼规则、写工具——同一条工作流跑得越多，质量越高、速度越快、成本越低

**核心理念**
- 传统 AI 产品卖"工具"，Solarian 交付"结果"
- 用户是董事长，Agent 是硅基员工
- 不是第 11 个 AI 工具，而是 AI 团队管理系统

**差异化 vs 竞品**
| 维度 | Zapier/n8n/Dify | Solarian |
|------|----------------|----------|
| 目标用户 | 开发者 / 技术团队 | 非技术一人公司 |
| Agent 身份系统 | 无或弱 | IDENTITY + SOUL + MEMORY |
| 协作模式 | 广播 / 触发器 | 流水线顺序交接（DAG） |
| 记忆持久化 | 无或云端 | 本地文件系统，跨 Session |
| 自进化能力 | 无 | 自写 Skill + 规则 + 工具 |
| 设计调性 | 工具化，偏开发者 | 专业温暖，Glassmorphism |

**竞品定位空白**
Solarian 占据"低技术门槛 × 深度多 Agent 协作"象限——当前无主要竞品。

### 竞品格局分析

| 竞品 | 规模 / 营收 | 核心局限 | 战略启示 |
|------|------------|---------|---------|
| Zapier | 3M+ 用户，$310M 营收（2024） | 触发器-动作模型，无 Agent 概念 | 非技术用户愿付费，$20/月 可接受 |
| Make | 开源友好 | 技术型用户，非技术门槛高 | 复杂流程有需求，但用户群不同 |
| n8n | 开源，AI 工作流有市场，估值 $25 亿 | 明确定位开发者，非技术用户门槛极高 | 开发者与非技术用户之间存在巨大鸿沟 |
| Dify | Pro $59/月 | 聚焦 AI 应用开发，非多 Agent 协作 | 开发者愿为 AI 工具付费 |
| Coze（字节） | 企业级 | 面向企业，个人 Solopreneur 是空白 | 字节入场证明 Agent 平台战略价值 |
| Claude Code | $10 亿 ARR | 会话结束团队解散，无持久化，多 Agent 成本 15 倍 | Solarian 定位于其上层，提供持久化管理层 |

**Claude Code 的结构性局限（竞品分析关键数据点）**
- AI 生成代码 PR 被拒率 **67.3%**，人类代码仅 15.6%——缺乏经验积累是根因
- 多 Agent 模式成本高达标准用量 **15 倍**
